# sheeprunner

A mobile game where you are a running sheep. You enter into and compete against other runners, that are other similar anthropomorphic animals. You can win competitions, and go on to the next levels. The runs are all many miles - start at mile run, then 5k, then 10k, then half-marathon, then full marathon, then some 75 mile, 100 mile, and 200 mile ultras - these should follow real running events in real places all over the world. 

The running mechanic is simple, once the race begins you steer the runner with your finger, and collect refreshments like pickle juice along the way, and also bass / drum and bass music events, so you run by speakers, and then mountains. 

There will be a soundtrack playing, I will put mp3s into a soundtrack/ folder.

The game should be 3D, fun, engaging, and really great. 

Use Phaser or a similar game engine to make this easy.
